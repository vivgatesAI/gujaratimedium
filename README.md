## 🎯 Course Topics (8 Modules)

1. **AI Agents** - Autonomous goal-directed systems
2. **Multi-Agent Swarms** - Collaborative intelligence
3. **Agentic Harnesses** - Orchestration frameworks
4. **Context Engineering** - Prompt & memory optimization
5. **Agentic Engineering** - Production-ready agent patterns
6. **MCP Servers** - Model Context Protocol for tool access
7. **A2A Communication** - Agent-to-Agent protocols
8. **AI Tech Stack** - Complete modern AI landscape

## 🛠️ Hands-On Exercises

5 progressive projects:
1. First autonomous agent (LangChain + Serper API)

4. A2A peer-to-peer network
5. Production AI platform deployment

## 🎨 Art Style Inspirations

- **Pithora** - Ceremonial wall paintings with horse-riding deities
- **Warli** - Stick-figure narratives with geometric patterns
- **Rogan** - Castor-oil based black-background vibrant art
- **Mata ni Pachedi** - Temple cloth paintings with religious motifs
- **Kutch Embroidery** - Mirror work, vibrant textiles, peacock motifs

## 🚀 How to Use

1. Open `index.html` in any modern browser
2. Navigate through topics using the fixed navigation bar
3. Complete interactive quizzes to test understanding
4. Follow the timeline exercises for hands-on learning
5. The site is fully responsive (mobile/tablet/desktop)
