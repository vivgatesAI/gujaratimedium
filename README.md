## 🎨 Art Style Inspirations

- **Pithora** — Ceremonial wall paintings with horse-riding deities
- **Warli** — Stick-figure narratives with geometric patterns
- **Rogan** — Castor-oil based black-background vibrant art
- **Mata ni Pachedi** — Temple cloth paintings with religious motifs
- **Kutch Embroidery** — Mirror work, vibrant textiles, peacock motifs

## 🚀 How to Use

1. Open `index.html` in any modern browser, or visit the live site
2. Navigate through topics using the fixed navigation bar
3. Complete interactive quizzes (5 questions) to test understanding
4. Track module progress with checkboxes (saved to localStorage)
5. Reference the Gujarati glossary for key terms
6. The site is fully responsive (mobile/tablet/desktop)
